## Note
/init Command Summary:

The /init command generates a CLAUDE.md file (prefixed with # CLAUDE.md as the main title) that documents your codebase for future Claude Code instances. It analyzes architecture, build/test commands, and key setup info. The # symbols are markdown headers (# = H1, ## = H2, etc.) that structure the documentation — so /init creates both the file and its organized layout.
